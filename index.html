SisCristóvão

Controle de Serviços e Agendamentos Práticos

Bem-vindo ao SisCristóvão 1.0

Uma plataforma didática desenvolvida para gerenciar a prestação de serviços, organizar a alocação de equipes e otimizar o fluxo de agendamentos dos alunos do São Cristóvão.

Este sistema conecta a interface Web ao banco de dados SQLite para simular um ambiente real de ordens de serviço e atendimento ao cliente.

🚀 Módulos Práticos do Sistema:

Módulo Descrição Operacional
👥 Clientes Cadastro de solicitantes e gerenciamento de contatos para assistência técnica.
🛠️ Serviços Catálogo de serviços disponíveis, incluindo escopo, prazos estimados e valores.
📅 Agendamentos Abertura de ordens de serviço, reserva de horários e alocação de alunos responsáveis.

📊 Estrutura Tecnológica

  • Ambiente Escolar: Práticas de laboratório na Unidade São Cristóvão.
  • Backend: Node.js com Express e API RESTful.
  • Banco de Dados: SQLite (`siscristovao.db`) com relacionamentos mestre-detalhe.
